# Metro Auto Detailing — Website Project

## Project Overview
Rebuilding the Metro Auto Detailing website from scratch in Next.js.
Current site is WordPress + Elementor hosted on Hostinger.
We are NOT editing WordPress. We are building a fresh Next.js site that mirrors the current structure but with cleaner components and better design.
When complete, metroautodetailing.pro will point to Vercel instead of Hostinger.

## Workflow
- Work section by section, one session per section
- Always read CLAUDE.md first for context
- Reference brand.md for all design decisions
- Reference site-overview.md for structure and goals
- Reference sections.md for what each section needs

## Tech Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Components: shadcn/ui + 21st.dev via magic MCP
- Hosting: Vercel
- Contact Form: Formspree
- Fonts: Montserrat (primary), DM Sans (prices/numbers)

## MCP Tools Available
- obsidian-vault: read/write Metro-Vault files
- shadcn: component registry
- magic: 21st.dev component library
- context7: live Next.js and Tailwind docs
- chrome-devtools: browser inspection and debugging
